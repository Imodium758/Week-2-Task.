Semantic html are divided into two categories based on their usage. Which are:

  • HTML semantic tags for structure
  • HTML semantic tags for text

Html semantic tags for structure consists of the following;

  1. Header
  2. Nav
  3. Main
  4. Article
  5. Section
  6. Aside
  7. Footer

Html semantic tags for text consists of the following;

  1. H1
  2. H2 to H6
  3. P
  4. A
  5. ol
  6. ul
  7. q
  8. em
  9. Strong
  10. Code and many more...
